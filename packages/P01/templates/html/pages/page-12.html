十一、精准干预方案： 饮食与营养

以12周为周期，逐步修复微生态与代谢稳态

饮食原则

  • 多样化：保证食物多样，摄入丰富植物性食物

  • 高纤维：增加膳食纤维，促进有益菌生长

  • 抗炎饮食：减少加工食品与添加糖，减轻慢性炎症

  • 优质营养：保证优质蛋白与健康脂肪摄入

营养重点

膳食纤维 ≥ 25-30 g / 天

以可溶性纤维为主（如菊粉、燕麦等）

植物多酚与抗氧化物

多吃彩色蔬果、深色叶菜、浆果类

优质蛋白

鱼类、豆类、蛋类、发酵乳制品为主

健康脂肪

橄榄油、坚果、牛油果、深海鱼类

每日饮食搭配建议

早餐 07:00-09:00

高纤谷物 + 优质蛋白

提供稳定能量，唤醒代谢

  • 燕麦 / 全麦面包
  • 鸡蛋 / 无糖酸奶
  • 坚果 / 蓝莓
午餐 12:00-13:30

多色蔬菜 + 豆类 + 全谷物

丰富膳食纤维，平衡血糖

  • 多色蔬菜（至少 3 种颜色）
  • 豆类 / 豆制品
  • 糙米 / 藜麦 / 全麦饭
晚餐 18:00-19:30

控制精制碳水 + 优质脂肪

减轻夜间代谢负担，促进修复

  • 深绿叶菜 / 清炒蔬菜
  • 优质脂肪（橄榄油 / 牛油果）
  • 少量全谷物 / 根茎类（可选）

建议优先食物清单

燕麦
菊粉
豆类
深绿叶菜
蓝莓
发酵乳制品
鱼类

温馨提示： 优食干预需长期坚持并结合个人情况调整，建议与专业营养师制定个性化方案。
