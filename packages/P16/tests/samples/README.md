当前版本已接入 P16《药物基因组学评估健康管理报告》HTML 模板、关键字段配置、规则草案、OCR 策略占位和 AI 提示词。

当前已基于以下样例完成 P16 OCR 策略 v0.2 升级：

- `D:\AWK-OCR\功能医学检测报告模板（2026.4.3）\P16\hebing44831.pdf`
- `D:\AWK-OCR\功能医学检测报告模板（2026.4.3）\P16\OCR.txt`

本版本支持：

- 直接导入 `reports[]` 聚合 JSON 结构
- 解析样例 PDF 时优先读取同目录 `OCR.txt` 作为结构化 OCR sidecar
- 输出高血压、他汀、CYP2C19、阿司匹林、替格瑞洛、高血糖和静脉血栓七类核心结果
